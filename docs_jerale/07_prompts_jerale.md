# Bitácora IA

## Objetivo

Registrar de forma realista cómo se utilizó la inteligencia artificial para apoyar el proceso de documentación del laboratorio.

## Procedimiento realizado

1. Prompt inicial: "Ayúdame a redactar una wiki técnica sencilla sobre un laboratorio de Windows Server en VirtualBox." 
   - La respuesta fue útil, pero estaba demasiado general. Se corrigió manualmente para que sonara más técnica y más cercana al trabajo realizado.

2. Prompt de mejora: "Haz el texto más claro y ordenado, con secciones como Objetivo, Procedimiento realizado, Resultado obtenido y Evidencia." 
   - Se ajustó la estructura para que cada página tuviera el formato solicitado.

3. Prompt para contenido específico: "Escribe un apartado sobre Active Directory usando el dominio inacap.local y el servidor SRV-DC01." 
   - Se corrigió manualmente el lenguaje para que no pareciera demasiado genérico y para que reflejara el contexto real del laboratorio.

4. Prompt para revisar estilo: "Quiero que el texto se vea como una nota de estudiante, no como un manual corporativo." 
   - Se cambió el tono para que fuera más natural y menos formal.

5. Prompt final: "Genera una bitácora de uso de IA con 5 prompts y una reflexión corta." 
   - Se corrigió manualmente la redacción para que fuera más honesta y menos robotizada.

## Resultado obtenido

La documentación quedó más clara, con una estructura consistente y un tono adecuado para una wiki estudiantil.

## Evidencia

No fue necesaria una captura de pantalla para esta sección, pero sí se registró el proceso de revisión y corrección manual.

Se corrigió manualmente el contenido de los prompts para evitar frases demasiado artificiales, para ajustar detalles técnicos y para mantener coherencia con el laboratorio real. La IA ayudó con la base del texto, pero la revisión final fue hecha por mí.

Reflexión corta: usar IA puede acelerar la escritura, pero siempre es necesario revisar que la información sea clara, correcta y coherente con el contexto.
