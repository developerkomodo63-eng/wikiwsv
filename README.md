# Wiki Jerale

Esta es una wiki simple en React y Vite para documentar un laboratorio de Windows Server en VirtualBox.

## Instalación

```bash
npm install
npm run dev
```

## Despliegue en Vercel

1. Sube este repositorio a GitHub.
2. Crea un proyecto nuevo en Vercel y conéctalo al repositorio.
3. Vercel detectará automáticamente Vite y realizará el despliegue.
4. La aplicación quedará disponible en una URL pública.
